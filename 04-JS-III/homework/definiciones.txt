Arrays

Un array es una coleccion ordenada de datos
Particularmente en Javascript el array puede componerse de cualquier tipo de dato, incluyendo enteros, caracteres, cadenas de caracteres, booleanos, indefinidos y hasta de otros strings.
Se accede a cada elemento con un subindice.
Cada elemento string es a su vez un array.
